짱멋진 보물섬 자동채점기
========

이쁘고 보기 쉬운 PL 4번과제 자동 채점기

```console
curl https://raw.githubusercontent.com/simnalamburt/snucse.pl/master/hw4/test -O
ocaml test
```

디버깅할 수 있도록 해줍니다.
